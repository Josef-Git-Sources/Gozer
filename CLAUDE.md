# גוזר (gozer.io) — הקשר לפיתוח עם Claude

## על הפרויקט
פלטפורמת ווב לחיילים משוחררים בישראל.
- **Stack:** Next.js 14 + TypeScript + Tailwind + PostgreSQL (Supabase) + Prisma + NextAuth
- **שפה:** עברית מלאה, RTL
- **דומיין:** gozer.io
- **GitHub:** github.com/Josef-Git-Sources/Gozer
- **גישה לתוכן:** תוכן כללי פתוח לכולם | מידע אישי דורש login

## מודולים
1. מרכז זכויות וחובות — מאמרים, קטגוריות, SEO
2. צ'קליסט שחרור — משימות לפי תקופות
3. מתכנן טיול — מחשבון עלויות לפי יעד ומשך
4. הכוונה מקצועית — שאלון נטיות + המלצות קורסים
5. פאנל ניהול — ניהול תוכן, משתמשים, סטטיסטיקות

## ארכיטקטורה
```
src/
├── app/
│   ├── admin/          # פאנל ניהול
│   ├── api/            # API routes
│   ├── checklist/      # צ'קליסט שחרור
│   ├── kariera/        # הכוונה מקצועית
│   ├── login/          # כניסה
│   ├── profile/        # פרופיל משתמש
│   ├── register/       # הרשמה
│   ├── tiyul/          # מתכנן טיול
│   └── zchuyot/        # זכויות וחובות
├── components/layout/  # Navbar, Footer, Providers
├── lib/                # auth.ts, prisma.ts, utils.ts
└── types/              # next-auth.d.ts
prisma/
├── schema.prisma       # מבנה DB מלא
└── seed.ts             # נתוני ברירת מחדל
```
